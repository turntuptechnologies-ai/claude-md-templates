# claude-md-templates — CLAUDE.md テンプレート生成ツール

このリポジトリは Claude Code で使う CLAUDE.md のテンプレート集です。
あなた（Claude Code）は、ユーザーの指示に基づいて対象プロジェクト用の CLAUDE.md を生成する役割を担います。

## あなたの役割

ユーザーから「○○プロジェクト用の CLAUDE.md を作って」と依頼されたら、以下の手順で対応してください。

### 1. テンプレートを読む

- `templates/base.md` — 全プロジェクト共通のベーステンプレート
- `templates/agent-teams.md` — Agent Teams 用のチーム体制・作業方式テンプレート

### 2. プロジェクト情報をヒアリングする

テンプレート内の `{{ ... }}` 記入欄を埋めるために、ユーザーに以下を質問してください:

- プロジェクト名と概要（何を作るのか）
- 技術スタック（言語・フレームワーク・DB 等）
- ビルド・テスト・リントのコマンド
- コーディング規約（言語固有のルール）
- 特別な制約事項（あれば）

すでにリポジトリが存在する場合は、コードを読んで自動的に情報を収集し、不足分だけ質問してください。

### 3. CLAUDE.md を生成する

テンプレートの `{{ ... }}` 部分をヒアリング結果で置き換え、プロジェクトに合った CLAUDE.md を生成してください。

- 不要なセクションは削除する（例: Dependabot を使わないなら該当セクションを省略）
- プロジェクト固有の情報を適切に追加する
- `examples/` ディレクトリに記入済みの例があるので参考にする

### 4. 出力する

生成した CLAUDE.md を対象プロジェクトのルートディレクトリに配置してください。
対象プロジェクトのパスが不明な場合は、ユーザーに確認してください。

## 言語ルール

- ユーザーとのやり取り、生成する CLAUDE.md の内容は、ユーザーの言語に合わせる
- デフォルトは日本語
